# MVP Roadmap (First Casual Game)

## Phase 1: Playable Prototype (Days 1-2)
- Implement single mechanic (one-tap)
- Add win/lose condition
- Add restart
- Validate 60 FPS and stable loop

## Phase 2: Juice (Days 3-5)
- Add particles
- Add haptics
- Add SFX
- Add score popups and transition animations

## Phase 3: Meta Layer (Days 6-8)
- Best score persistence
- Simple coin economy
- Unlock one cosmetic skin

## Phase 4: Polish + Release Prep (Days 9-14)
- Onboarding tooltip
- App icon + screenshots
- Crash tracking + analytics
- TestFlight build and external testing
