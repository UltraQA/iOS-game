# iOS Game Starter

Starter foundation for a first casual iOS game on `Swift + SpriteKit`.

## Current Status

- Project structure created
- Architecture and roadmap documented
- Design system tokens prepared for SwiftUI/UIKit mapping
- Starter Swift files scaffolded

## Next Immediate Goal

Implement a playable one-tap prototype (single screen, 60-second gameplay loop, lose/restart).

## Structure

- `Docs/Architecture` - architectural decisions and module boundaries
- `Docs/DesignSystem` - color/typography/spacing/components/screen layouts
- `Docs/Roadmap` - staged implementation plan
- `iOSGame` - source structure ready for SpriteKit integration
- `Tests` - unit and snapshot test targets layout
